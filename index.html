CSS Propiedades y Valores

Propiedad/Valor Descripción y notas

background

Cualquier combinación de los valores para: background-attachment, background-color, background-image, background-repeat y/o background-position o inherit

Para cambiar el color o imagen de fondo de los elementos

El valor inicial depende de las propiedades individuales; no se hereda

background-attachment

scroll, fixed o inherit

Para determinar si debería scrollear la imagen y cómo hacerlo

El valor inicial es scroll; no se hereda

background-color

Cualquier color, transparent o inherit

Para establecer un color de fondo a un elemento

El valor inicial es transparent; no se hereda

background-image

URL, none o inherit

Para establecer una imagen de fondo a un elemento

El valor inicial es none; no se hereda

background-position

Toma 1 o 2 valores.
El primero puede ser %, unidad de medida, top, center o bottom.
El segundo puede ser %, unidad de medida, left, right o center.
O puede ser inherit

Para establecer la posición física de un background-image específico

El valor inicial es 0% 0%; si se introduce 1 valor este será para la posición horizontal, la posición vertical tomará como valor 50% (o center); no se hereda; cuando utilizamos % nos referimos al tamaño de la propia caja

background-repeat

Puede ser repeat, repeat-x, repeat-y, no-repeat o inherit

Para establecer si debe repetirse una imagen para rellenar el espacio, y cómo debe hacerlo

El valor inicial es repeat; no se hereda

border

Puede ser cualquier combinación de los valores de border-width, border-style, y/o border-color, o inherit

Para definir todos los aspectos del borde de un elemento

El valor inicial es depende de propiedades individuales; no se hereda

border-color

Puede ser de 1 a 4 colores, transparent o inherit

Para definir los color de los bordes de un elemento

El valor inicial es el valor de la propiedad color; no se hereda

border-spacing

Puede sero 1 o 2 medidas de largo o inherit

Para definir lel espacio entre los bordes de una tabla

El valor inicial es 0; debería ser sólo aplicado a los elementos de una tabla; no se hereda

border-style

Puede ser de 1 a 4 de los siguientes valores: none, dotted, dashed, solid, double, groove, ridge, inset, outset, o inherit

Para definir sólo el estilo de un borde o uno o más lados de un elemento

El valor inicial es none; no se hereda

border-* (*: top, right, bottom, left)

Cualquier combinación de valores de: border-width, border-style y/o border-color (1 de cada uno), o inherit

Para definir las 3 propiedades de un único borde a la vez

El valor inicial depende de valores individuales; no se hereda

border-*-color (*: top, right, bottom, left)

Puede ser un color o inherit

Para definir el color de un único borde de un elemento

El valor inicial es el valor de la propiedad color; no se hereda

border-*-style (*: top, right, bottom, left)

Puede ser de 1 de los siguientes valores: none, dotted, dashed, solid, double, groove, ridge, inset, outset, o inherit

Para definir el estilo de un único borde de un elemento

El valor inicial es el valor de la propiedad color; no se hereda

border-*-width (*: top, right, bottom, left)

Puede ser de 1 de los siguientes valores: thin, medium, thick o una medida de largo

Para definir el ancho de un único borde de un elemento

El valor inicial es medium; no se hereda

border-width

Puede ser de 1 a 4 valores de los siguientes valores: thin, medium, thick o una medida de largo

Para definir el ancho de 1 o más bordes de un elemento

El valor inicial es medium; no se hereda

bottom

Puede ser: %, una medida de largo, auto o inherit

Para definir la distancia que un elemento deberia alejarse del borde de abajo de su elemento padre

El valor inicial es auto; cuando usa % toma como referencia el alto del bloque contenedor

clear

Puede ser: none, left, right, both o inherit

Para evitar que los elementos floten en uno o ambos lados de un elemento

El valor inicial es none; deberia sólo ser aplicado a elementos de tipo bloque; no se hereda

color

Puede ser un color o inherit

Para establecer el color de primer plano de un elemento o la fuente

El valor inicial es inherit; deberia sólo ser aplicado a elementos de tipo bloque; se hereda

cursor

Puede ser: auto, crosshair, default, pointer, move, e-resize, ne-resize, nw-resize, n-resize, se-resize, sw-resize, s-resize, w-resize, text, wait, help, a URL, o inherit

Para establecer la forma del cursor

El valor inicial es auto; se hereda

display

Puede ser: inline, block, list-item, none, o inherit

Para establecer si un elemento debe ser mostrado y cómo hacerlo

El valor inicial es inline/block; no se hereda

float

Puede ser: inline, block, list-item, none, o inherit

Para determinar en qué lado de un elemento pueden flotar otros elementos

El valor inicial es none; no debería ser aplicado a elementos posicionados o contenido generado; no se hereda

font-family

Puede ser 1 o más nombres de fuentes, poniendo al último una genérica, o inherit

Para elegir la fuente (family) de nuestro texto

El valor inicial depende del navegador; se hereda

font-size

Puede ser una medida de longitud (relativa o absoluta), % o inherit

Para establecer el tamaño de nuestro texto

El valor inicial es medium; se hereda; si utilizamos porcentajes este se tomará con respecto al tamaño de la fuente del elemento padre

font-style

Puede ser normal, italic, oblique, o inherit

Para convertir el texto a italic

El valor inicial es normal; se hereda

font-variant

Puede ser normal, small-caps, o inherit

Para convertir el texto en Small Caps

El valor inicial es normal; se hereda

font-weight

Puede ser: normal, bold, bolder, lighter, 100, 200, 300, 400, 500, 600, 700, 800, 900, o inherit

Para aplicar, remover y ajustar el formato de negrita del texto

El valor inicial es normal; se hereda

height

Puede ser: una medida de longitud, %, auto, o inherit

Para aplicar una altura a un elemento

El valor inicial es auto; debería ser aplicado a todos los elementos excepto los elementos en línea, las columnas de las tablas y a los grupos de columnas; no se hereda

left

Puede ser: una medida de longitud, %, auto, o inherit

Para establecer la distancia que un elemento debería estar alejado del borde izquierdo de su elemento padre

El valor inicial es auto; debería ser aplicado sólo a los elementos posicionados; no se hereda; si utilizamos porcentajes este se tomará con respecto al ancho del elemento que lo contiene

letter-spacing

Puede ser: una medida de longitud, normal, o inherit

Para establecer la cantidad de espacios entre los caracteres del texto

El valor inicial es normal; se hereda

line-height

Puede ser: una medida de longitud, un número, %, normal, o inherit

Para establecer la cantidad de espacios entre las líneas del texto

El valor inicial es normal; se hereda; si utilizamos porcentajes este se tomará con respecto al tamaño de la fuente del mismo elemento

list-style

Cualquier combinación de valores para: list-style-type, list-style-position y/o list-style-image, o inherit

Para establecer una marca de item-list y su posición

El valor inicial depende de los valores iniciales de los elementos individuales; sólo se aplica a elementos de tipo lista; se hereda

list-style-image

Puede ser: URL, none, o inherit

Para establecer una marca de item-list personalizada para una lista

El valor inicial es none; sólo se aplica a elementos de tipo lista; sobrescribe a list-style-type; se hereda

list-style-position

Puede ser: inside, outside, o inherit

Para establecer la posición de una marca de item-list

El valor inicial es none; sólo se aplica a elementos de tipo lista; se hereda

list-style-type

Puede ser: disc, circle, square, decimal, lower-roman, upper-roman, lower-alpha, upper-alpha, none, o inherit

Para establecer un tipo de marca de item-list para una lista

El valor inicial es disc; sólo se aplica a elementos de tipo lista; no usar si usamos list-style-image (sin verificar); se hereda

margin

Puede ser de 1 a 4 valores de los siguientes valores: una medida de longitud, %, auto, o inherit

Para establecer la cantidad de espacio entre uno o más lados del borde de un elemento y su elemento padre o hermanos adyacentes

El valor inicial depende del navegador y del valor de su ancho; no se hereda; cuando usa % toma como referencia el ancho del elemento que lo contiene

margin-* (*: top, right, bottom, left)

Puede ser: una medida de longitud, %, auto, o inherit

Para establecer la cantidad de espacio entre uno de sus lados del borde de un elemento y su elemento padre o hermanos adyacentes

El valor inicial es 0; no se hereda; cuando usa % toma como referencia el ancho del elemento que lo contiene; los valores de margin-left y margin-right pueden ser anulados si la suma del ancho y ambos márgenes es mayor al ancho del elemento padre contenedor

max-height, max-width

Puede ser: una medida de longitud, %, auto, o inherit

Para establecer el alto/ancho máximo de un elemento

El valor inicial es none; no se hereda; cuando usa % toma como referencia el alto/ancho del elemento que lo contiene; no debería ser aplicado a elementos inline o elementos de tabla

min-height, min-width

Puede ser: una medida de longitud, %, auto, o inherit

Para establecer el alto/ancho mínimo de un elemento

El valor inicial es 0; no se hereda; cuando usa % toma como referencia el alto/ancho del elemento que lo contiene; no debería ser aplicado a elementos inline o elementos de tabla

overflow

Puede ser: visible, hidden, scroll, auto, o inherit

Para establecer como será tratado el contenido que desborda el área de contenido del elemento

El valor inicial es visible; no se hereda; debería sólo ser aplicado a elementos de tipo block

padding

Puede ser de 1 a 4 valores de los siguientes valores: una medida de longitud, %, o inherit

Para establecer la cantidad de espacio entre uno de sus lados del contenido de un elemento y su borde

El valor inicial depende del navegador; no se hereda; cuando usa % toma como referencia el ancho del elemento que lo contiene

padding-* (*: top, right, bottom, left)

Puede ser: una medida de longitud, %, o inherit

Para establecer la cantidad de espacio entre uno de sus lados del contenido de un elemento y su borde

El valor inicial es 0; no se hereda; cuando usa % toma como referencia el ancho del elemento que lo contiene

position

Puede ser: static, relative, absolute, fixed, o inherit

Para establecer cómo un elemento debería ser posicionado con respecto al flujo del documento

El valor inicial es static; no se hereda; no debería ser aplicado a contenido generado

right

Puede ser: una medida de longitud, %, auto, o inherit

Para establecer la distancia que un elemento debería estar alejado del borde derecho de su elemento padre

El valor inicial es auto; debería ser aplicado sólo a los elementos posicionados; no se hereda; si utilizamos porcentajes este se tomará con respecto al ancho del elemento que lo contiene

text-align

Puede ser: left, right, center, justify, a string, o inherit

Para alinear texto

El valor inicial depende del navegador y la dirección de escritura; se hereda; debería sólo ser aplicado a elementos de tipo block

text-decoration

Cualquier combinación de: underline, overline, line-through, and blink, o none, o inherit

Para decorar texto

El valor inicial es none; no se hereda

text-indent

Puede ser: una medida de longitud, %, o inherit

Para establecer la cantidad de espacio que la primera línea de un párrafo debería estar indentado

El valor inicial es 0; se hereda; debería sólo ser aplicado a elementos de tipo block; si utilizamos porcentajes este se tomará con respecto al ancho del elemento que lo contiene

text-transform

Puede ser: capitalize, uppercase, lowercase, none, o inherit

Para poner en mayúscula la letra inicial de cada palabra en un texto

El valor inicial es none; se hereda

top

Puede ser: una medida de longitud, %, auto, o inherit

Para establecer la distancia que un elemento debería estar alejado del borde de arriba de su elemento padre

El valor inicial es auto; debería ser aplicado sólo a los elementos posicionados; no se hereda; si utilizamos porcentajes este se tomará con respecto al alto del elemento que lo contiene

visibility

Puede ser: visible, hidden, collapse, o inherit

Para ocultar o no un elemento sin afectar el flujo del documento

El valor inicial es inherit; no se hereda

white-space

Puede ser: normal, pre, nowrap, o inherit

Para especificar como debería ser tratado el espacio en blanco

El valor inicial es normal; debería sólo ser aplicado a elementos de tipo block; se hereda

width

Puede ser: una medida de longitud, %, auto, o inherit

Para aplicar el ancho a un elemento

El valor inicial es auto; no debería ser aplicado a los elementos inline, las filas de las tablas y a los grupos de filas; no se hereda; si utilizamos porcentajes este se tomará con respecto al ancho del elemento que lo contiene

word-spacing

Puede ser: normal, una medida de longitud, o inherit

Para especificar la distancia entre palabras

El valor inicial es normal; se hereda

z-index

Puede ser: auto, un entero, o inherit

Para especificar la profundidad de un elemento con respecto a los elementos que se superponen

El valor inicial es auto; no se hereda; debería sólo aplicarse a elementos posicionados
