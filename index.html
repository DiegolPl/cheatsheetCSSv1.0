CSS Propiedades y Valores

Propiedad/Valor Descripción y notas

background

Cualquier combinación de los valores para: background-attachment, background-color, background-image, background-repeat y/o background-position o inherit

Para cambiar el color o imagen de fondo de los elementos

El valor inicial depende de las propiedades individuales; no se hereda

background-attachment

scroll, fixed o inherit

Para determinar si debería scrollear la imagen y cómo hacerlo

El valor inicial es scroll; no se hereda

background-color

Cualquier color, transparent o inherit

Para cambiar el color de fondo de un elemento

El valor inicial es transparent; no se hereda
